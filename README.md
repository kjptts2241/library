<br/>


멤버 구성<br/>

|이름|역할|기능 및 구현|
|:---:|:---:|:---|
|박동준|백엔드|도서관 지도 기능, 내 서제 기능|
|김동영|백엔드|검색 기능, 로그인 및 회원가입, 내 서재 기능, AWS 서버 구축|

|김종율|프론트 엔드|로그인 및 회원가입 페이지|
